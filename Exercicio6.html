Exemplo 6 HTMK - Uso das tags <p>, <hr>,<br>,<strong>,<em>

Este exemplo demonstra o uso das tags de formatção de texto bem como sua organização visual.

Descrição das tags abordadas

  • <p> : Define um parágrafo
  • <hr> : Cria uma linha horizontal para separar conteúdos
  • <br> : Insere uma quebra de linha no documento html
  • <strong> : Define texto em negrito (com importância semântica)
  • <em> : Define texto em itálico também com importância semântica
