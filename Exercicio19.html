Visao geral sobre CSS

CSS é um acrônimo para folhas de estilo em cascata, essa técnica visa personalizar e estilizar um documento em HTML, para isso usaos regras css que representam uma estrutura que define como os elementos selecionados devem ser estilizados. Essa regra é composta pelas seguintes partes:

  • Seletor:Seletor indica quais elementos serão afetados pelo estilo definido
  • Bloco de declaraçãoÉ o bloco composto por propriedades e valores, esse bloco será aplicado ao elemento HTML.

Tipos de seletores

Em CSS seletores são padrões usados para selecionar e estilizar os elementos HTML

Seletores de Classe

Os seletores de classe em CSS sao aqueles que sao aplicados a elementos HTML que possuam os atributos class (.). Esse seletor é utilizado amplamente quando queremos estilizar um documento HTML.

Este seletor sempre começará com um pont seguido pelo nome do seletor e suas propriedades e valores.

Neste exemplo usamos seletores de classe e de tipo ara estilizar um documento HTML.
