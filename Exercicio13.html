Exemplo 13 html - Formulários

Os formulários HTML são uma das principais formas de se interagir com aplicações web. Esses Formulários utilizam diversos componentes que melhorarão a experiência do usuários.

Para trabalhar com formulários devemos usar a tag <form> em conjunto com outras tags que vão representar os componentes de formulario, tais como, lgabels(rótulos) inputs(caixas de texto), botões, elementos de checagem, de seleção entre outros componentes. Neste exemplo, demonstraremos o uso dos métodos GET e Post

Metodos do Protocolo HTML

GET

Quando trabalhamos com a criação de formulários HTML, nós deveremos a maior parte das vezes utilizar o protocolo de rede HTTP, esse protocolo possui vários métnodos que são utilizados para facilitar o processo de comunicação entre o cliente e o servidor.

O método GET é aquele que anexa os dados do formulário a uma url, esses dados dos formulários são visíveis na barra de endereço no navegador.

É importante saber que este método não é recomendado para operações de inclusão, pois os dados vão aparecer para qualquer pessoa, o que pode ser inseguro.

Post

Ja o método POST que envia os dados para o servidor no corpo da página. Essas informações não são visíveis para o cliente sendo um pouco mais seguras que quando utilizamos o método GET.
