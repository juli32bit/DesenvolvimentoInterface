Exemplo 4 HTML - Estrutura geral de um documento HTML

Este exemplo mostra a estrutura mínima de um documento hmtl. Aqui estão as principais tags utilizadas:

  • <!DOCTYPE hmtl >: Define o tipo de documento HTML, ou seja a versão do html que sera usada na página.
  • <html >:A tag html é o elemento principal do documento hmtl ela define o container raiz de toda página.
  • <head >:A tag head contém os metadados que serão interpretados pelo navegador. Nela encontraremos o título da página e links e arquivos para configuração do documento.
  • <body >:A tag body é responsável pela exibição do conteúdo visível do documento, ou seja, tudo que estiver neste bloco será visível para o usuário.
