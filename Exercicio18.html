Visao geral sobre CSS

Css éum acrônimo para folhas de estilo de cascata, essa técnica visa personalizar e estilizar um documento HTML, para isso utilizamos regras CSS que representam uma estrutura que define como os elementos devem ser estilizados. Essa regra é composta pelas seguintes partes:

  • Seletor:Seletor indica quais elementos serão afetados pelo estilo definido
  • Bloco de declaração: É o bloco composto por propriedades e valores, esse bloco será aplicado ao elemento HTML.
