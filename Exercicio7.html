Exemplo 7 - HTML - Listas Ordenadas <ol> e listas não Ordenadas <ul>

Este expemplo mostra como criar listas ordenadas e não ordenadas, incluindo subitens(listas alinhadas).

Listas Ordenadas <ol>p

As listas ordenadas são aquelas que seguem uma ordem numérica ou por letras são utilizadas para mostrar conteúdo html em formato de lista.

Exemplo da lista ordenada <ol> juntamente com os elementos dessa lista é representado pela tag <li>

  1. item 1
  2. item 2
  3. item 3
  4. item 4
  5. item 5

Exemplo 2 de listas ordenadas

  1. item 1
    1. subitem 1.1
    2. subitem 1.2
    3. subitem 1.3
    4. subitem 1.4
    5. subitem 1.5
  2. item 2
  3. item 3
  4. item 4
    1. subitem 1.2
    2. subitem 2.3
    3. subitem 3.4
    4. subitem 4.5
  5. item 5
