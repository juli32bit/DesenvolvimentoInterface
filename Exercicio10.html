Exemplo 10 HTML - Tableas

As tabelas em HTML são representadas pela tag <table> que são uteis para exibir dados tabulares, tais como informações em linhas e colunas.

Descrição dos principais atributos para criação de tabelas em HTML.

  • <table>: Esta é a tag principal que envolve toda a tabela html.
  • <tr>: Esta tag representa uma linha da tabela e nela teremos as tags <th> e <td>
  • <th>: Esta tag representa o cabeçário da tabela, th é o acrônimo de tabele header.
  • <td>: A tag <td>(table data), representa a célula da tabela
  • <theader>: Tag semântica que representa o cabeçálho. Ela é utilizada juntamente com a tag <th>
  • <tbody>: Tag semântica que representa o corpo da tabela nela geralmente usamos a tag td
  • <tfoot>: Tag semântica que representa o rodapé da tabela.
