Visao geral sobre CSS

CSS é um acrônimo para folhas de estilo em cascata, essa técnica visa personalizar e estilizar um documento HTML. Para isso usamos regas CSS que representam uma estrutura que define como os elementos selecionados devem ser estilizados. Essa regra é composta pelas seguintes partes:

  • Seletor: Indica quais elementos serão afetados pelo estilo definido.
  • Bloco de declaração:É o bloco composto por propriedades e valores, esse bloco será aplicado ao elemento HTML.

Tipos de seletores

Em CSS seletores são padrões utilizados para selecionar e estilizar os elementos HTML em uma página web. Eles definem quais elementos receberão as regras de estilos que serão declaradas dentro das chaves. Possuímos vários tipos de seletores, sendo os principais:

Seletores de ID

O seletor de ID é aquele que aplica a regra CSS a um componente de HTML que tenha um atributo ID no componente HTML. Esse seletor de ID deve ser aplicado a um único elemento HTML, haja vista que o ID deve ser único. Este seletor tem prioridade sobre os demais.

Este seletor sempre começará com uma #, seguido pelo nome do seletor e suas propriedas e valores.
