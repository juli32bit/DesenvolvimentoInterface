Exemplo 5 HTML - Tags de Seção e Títulos

Este exemplo mostra como usar as tags de título <h1> à <h6> para organizar o conteúdo do documento html em seções hierárquicas

Descrição

As tags de título são usadas para definir cabeçalhos e subtítulos. Elas variam de <h1> (mais importante) à <h6> (menos importante).
