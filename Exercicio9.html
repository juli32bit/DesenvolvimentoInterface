Exemplo 9 - HTML

<header>

Exemplo sobre Web Semântica

Este bloco refere-se a tag semântica <header> que representa o cabeçário de uma página. Nele geralmente utilizaremos as tags de seção/título

<nav>

Neste bloco representamos a seção de navegação que utiliza a tag <nav>. Neste bloco definimos por exemplo os menus de navegção do nosso documento html. Por exemplo:

  • Início
  • Cadastro
  • Relatórios
  • Sobre
  • Movimentação

<main>

A tag <main> utilizada neste representa o contúdo principal da página. Nele teremos as tags semânticas <section>, <article> e <aside>.

HTML - Web Semântica

<section>

HTML - Web Semântica

Neste bloco estamos usando a tag semÇantica <section> para definir uma seção genérica de um documento.

A web semântica refere-se ao uso do HTML e outras tecnologias para estruturar o contéudo de uma página de forma que ela seja compreendida tanto por humanos quanto por máquinas. isso significa usar elementos HTML que descrevam o significado do contéudo, em vez de apenas sua aparência.

<article>

A tag semântica <article> defini um contpeudo independente, como uma postagem de blog ou notícia

<aside>

A tag semântica <aside> representa um contéudo auxiliar, relacionado, mas não essencial, ele muitas vezes é utilizado como um menu auxiliar

  • Menu auxiliar 1
  • Menu auxiliar 2
  • Menu auxiliar 3
  • Menu auxiliar 4
  • Menu auxiliar 5

<footer>

A tag semântica <footer> é aquela que define um rodapé do documento HTML. Nela geralmente colocamos informações de fechamento do nosso documento, tais como nome, autor, data, etc....
